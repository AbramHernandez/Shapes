# Shapes
Function reads shape descriptions(circle,triangle,square,point) to draw them on a 2-D grid. 
When the shapes are drawn to the 2-D it will print with Grid::print 
Ex: 
```
./shapes 

triangle 15 15 
square 0 0 24 
circle 1 1 
point 15 3 ? 
^D 

************************
* oo                   *
*o  o                  *

*                +     *
*               + +    *
*              +++++   *
*                      *
*                      *
*                      *
*                      *
*                      *
************************
```
